Lancement du projet:

Le projet à était fait avec docker afin de le démarrer il faut donc l'installé.
Sur windows 10 docker fonctionne avec une VM sur Hyper V,
Sur linux en revenche docker fonctionne directement.

Maintenant pour démarrer le projet il suffi de lancé une console dans le dossier du projet et d'écrire "docker-compose buid" et ensuite "docker-compose up"
